BranchGroup

Related Docs: object BranchGroup | package model

case class BranchGroup(sup: IdentifiedObject, maximumActivePower: Double, maximumReactivePower: Double, minimumActivePower: Double, minimumReactivePower: Double, monitorActivePower: Boolean, monitorReactivePower: Boolean) extends Element with Product with Serializable

A group of branch terminals whose directed flow summation is to be monitored. A branch group need not form a cutset of the network.
