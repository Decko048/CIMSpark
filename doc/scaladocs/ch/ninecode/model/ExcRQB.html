Related Docs: object ExcRQB | package model

final case class ExcRQB(sup: ExcitationSystemDynamics, ki0: Double, ki1: Double, klir: Double, klus: Double, lsat: Double, lus: Double, mesu: Double, t4m: Double, tc: Double, te: Double, tf: Double, ucmax: Double, ucmin: Double) extends Element with Product with Serializable

Excitation system type RQB (four-loop regulator, r?gulateur quatre boucles, developed in France) primarily used in nuclear or thermal generating units.

This excitation system shall be always used together with power system stabilizer type PssRQB.
