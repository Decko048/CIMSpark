ExcIEEEST6B

Related Docs: object ExcIEEEST6B | package model

case class ExcIEEEST6B(sup: ExcitationSystemDynamics, ilr: Double, kci: Double, kff: Double, kg: Double, kia: Double, klr: Double, km: Double, kpa: Double, oelin: String, tg: Double, vamax: Double, vamin: Double, vrmax: Double, vrmin: Double) extends Element with Product with Serializable

The class represents IEEE Std 421.5-2005 type ST6B model.

This model consists of a PI voltage regulator with an inner loop field voltage regulator and pre-control. The field voltage regulator implements a proportional control. The pre-control and the delay in the feedback circuit increase the dynamic response.
